# node-red-contrib-gira-endpoint


Integrate your Gira Homeserver devices with <b>Node-RED</b>

### Quick Start

Install with the built in <b>Node-RED Palette manager</b> or using npm:
```
npm install red-contrib-gira-endpoint
```
